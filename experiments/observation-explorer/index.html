Sentinel-2 GeoTIFF Timeline Explorer

This experiment loads Sentinel-2 COGs and renders RGB or NDVI layers. Add a Cesium ion token to enable high-resolution terrain and an API endpoint to supply your GeoTIFF catalog.

Paste the JSON array returned by your catalog API to skip the network request.

Token, API, and pasted JSON are saved in browser storage so you can resume after closing the tab. Use “Clear saved settings” to remove them.

Need the catalog API format? Review the GeoTIFF catalog API guide.
